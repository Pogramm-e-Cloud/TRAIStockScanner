# TRAI Stock Scanner

KI-gestützte Aktienanalyse- und Handelsplattform.

## Funktionen

- KI-Aktienanalyse
- Top 3 Kauf- und Verkaufssignale
- Charttechnik
- News-Auswertung
- Derivate-Rechner
- Warnsystem mit Sound
- Community-Chat
- Datenschutz & Login
- iOS, Android, Windows und macOS vorbereitet

## Plattformen

- Web
- Android
- iOS
- Windows
- macOS

## Sicherheit

Dieses Projekt nutzt:
- Firebase Authentication
- Firestore Datenbank
- geschützte API-Struktur
- keine öffentlichen API-Keys im Code
- Datenschutz- und AGB-Grundlagen

## Hinweis

Diese Software dient nur zu Analyse- und Informationszwecken.  
Keine Finanzberatung. Trading und Derivate können zum Totalverlust führen.
